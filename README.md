# Macintosh-S210H-EFI
Macintosh S210H EFI 黑苹果 EFI
S210 配置表
- **CPU**：酷睿®i5 i7 i9 标压 处理器(非低压)
- **硬盘**：1*M.2PCIE3.0x4+1*M2PCIE或Sata 2280双固态硬盘+1*2.5寸硬盘扩展HDD 三硬盘设计
- **内存**：双通道 笔记本DDR4 3200MHz Max 64G
- **无线**：2.4G/5G 双频WiFi6+蓝牙5.2(可更换天线)
- **有线**：双网口2.5G+1G RTL8125BG(仅i9)+i219V (i7为RTL8111H+i219V)
- **输出**：HDMI2.0a+DP 4K/60Hz (可转接VGA或DVI)
- **系统**：Win10/11 支持黑苹果 Linux Ubuntu
- **散热**：台式下压式铜芯散热风扇
- **尺寸**：134*144*64mm 1050g 全铝合金
- **电源**：整机90~100W 19V/4.7A TDP45W
- **接口**：音频耳麦二合一,Type-C (USB3.1 Gen1), USB3.0*4+USB2.0*2
- **支持**：软路由 Openwrt 虚拟机 ESXi/PVE 网络存储器 NAS… 定时开机√ 24小时运行√ 通电开机√ 网络唤醒√
*以上参数仅供参考，如有变动，以实物为准，详情请洽客服！*

- 除 WiFi 外全部可用，蓝牙可用。
